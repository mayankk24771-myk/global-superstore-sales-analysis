# 📊 Global Superstore Sales Analysis

## 📌 Problem Statement
Analyze global retail sales data to identify key trends, high-performing regions, and profit-driving products to support business decision-making.

## 📊 Dataset
- Global Superstore dataset
- Features: Sales, Profit, Quantity, Category, Region, Order Date

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

## 🔍 Analysis Performed
- Cleaned and transformed raw sales data
- Performed SQL queries to analyze:
  - Top-selling products
  - Regional performance
  - Profit trends
- Built interactive Power BI dashboard

## 📈 Key Insights
- Technology category generated highest revenue
- Certain regions showed high sales but low profit (cost issue)
- Seasonal spikes observed in Q4 sales
- Discounts negatively impacted profitability
